# Inventory-Analysis-Project
# 📦 Inventory Analysis Project – SQL Server + Power BI

## 🔹 Overview
This project demonstrates an end-to-end workflow for **inventory management and business intelligence** using SQL Server.  
It covers database schema design, data loading from CSV files, and analytical views for KPIs such as demand, inventory status, supplier performance, and returns.  
The SQL scripts are structured for clarity and modular execution.

---

## ⚙️ How to Run
1. **Schema Setup**  
   Run `SQL/01_schema.sql` → Creates the database, tables, and loads CSV data.

2. **Views Creation**  
   Run `SQL/02_views.sql` → Creates KPI views for demand, inventory, supplier performance, and returns.

---

## 📊 Views Created
- **v_demand_analysis** → Sales by product, category, region, customer type.
- **v_inventory_status** → Stock usage, average inventory, movement category, stock status.
- **v_supplier_performance** → Supplier lead times and order counts.
- **v_revenue_returns** → Revenue, gross margin, return rates by category.
- **v_sales_returns** → Sales units vs. returns, reason breakdown.
